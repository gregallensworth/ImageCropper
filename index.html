• Each image[type="file"] element will gain a new behavior, opening a image-cropping utility.
  • The size of this image-cropper panel (800x600 by default) determines the size of the cropped image, so the user is seeing an accurate representation of the final result.
  • When the user confirms with the "Crop & Save" button, the image[type="file"] is emptied and hidden, and a input[type="hidden"] field is created in its place, containing the base64-encoded data.
